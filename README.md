# 🏛️ SIMA-UNMARIS: Enterprise Asset Management & Survey System

## SIMA-UNMARIS (Sistem Informasi Manajemen Aset Universitas Stella Maris) adalah ekosistem digital terpadu yang dirancang untuk mengelola inventaris kampus, audit fisik (Stock Opname), hingga survei kepuasan civitas akademika. Sistem ini dikembangkan untuk meningkatkan transparansi dan efisiensi pengelolaan aset di lingkungan universitas.
